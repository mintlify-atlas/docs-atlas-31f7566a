UC3Tools is a free, open-source Windows Forms application that provides utilities for working with [Ultimate Coaster 3](https://store.steampowered.com/app/1999950/Ultimate_Coaster_3/) save files. Whether you're bringing in a coaster designed in No Limits 2's FVD++ tool, migrating a park from Ultimate Coaster 2, or building a custom terrain from a heightmap, UC3Tools handles the conversion.

## Key capabilities

- **FVD conversion** — Reads the tab-separated CSV output from FVD++ and reconstructs the track as a NURBS spline in UC3, with adjustable heartlining offset (−1.0 to +1.0 m).
- **UC2 migration** — Converts `.uc2sav` files to `.uc3world`, mapping terrain height data, water settings, camera position, 30+ coaster types, and all scenery objects.
- **Terrain generation** — Produces `.uc3world` files containing only a terrain mesh, optionally shaped from a greyscale PNG or JPG heightmap.
- **No Limits 2 round-trip** — Supports importing coasters designed using No Limits 2's FVD++ script export workflow.

## Technology

UC3Tools is built with:

- **.NET 9 / Windows Forms** — native Windows GUI
- **LibUC3** — reads and writes UC3's protobuf-based `.uc3world` binary format
- **LibUC2** — parses UC2's text-based UCON save format
- **VProto** — a custom protobuf serialization library written for this project
- **OpenTK.Mathematics** — 3D vector and quaternion math
